# 出行交通方式占比 · 大二学生一周时间分配全景

> 小组大主题「大二大学生一周时间分配全景」的子主题之一
> 子主题：**出行交通方式占比**（地铁 / 公交 / 步行 / 骑行 / 打车）

基于 D3.js v7 的环形饼图，按一周出行**次数**口径统计各交通方式的占比。

## 在线预览

启用 GitHub Pages 后访问：
`https://<你的用户名>.github.io/<仓库名>/`

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 主图（GitHub Pages 默认首页）。d3.pie 转角度 + d3.arc 转路径，环形布局，鼠标悬停在圆心显示次数与占比 |
| `transport_mode.csv` | 数据源。改这里的 `trips` 列即可更新图表 |

## 数据格式

```csv
id,mode,mode_en,trips,icon
1,地铁,Metro,12,🚇
...
```

`trips` 为一周内该方式的出行次数；占比由程序自动按总次数计算。
当前为**示例数据**，请替换为真实调研结果。

## 技术要点

- `d3.pie()` —— 原生数据转换为起止角度（startAngle / endAngle）
- `d3.arc()` —— 起止角度转换为 `<path>` 的 `d` 参数；`innerRadius > 0` 形成环图
- 内置备用数据：CSV 加载失败（如本地直接双击打开）时自动回退，保证图始终能渲染

## 本地运行

由于浏览器安全策略，直接双击打开 `index.html` 时 CSV 可能无法加载（会自动用内置数据渲染）。
若想本地读取真实 CSV，请起一个本地服务器：

```bash
python -m http.server
# 然后浏览器打开 http://localhost:8000/
```
